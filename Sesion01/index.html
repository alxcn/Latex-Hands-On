¿Qué es LaTeX?

Sistema de preparación de documentos de alta calidad.

  • No es un procesador de texto (como Word).
  • Enfoque WYSIWYM: What You See Is What You Mean.
  • Tú escribes contenido; LaTeX se encarga del diseño visual.

Ventajas para Ingeniería

  • Fórmulas matemáticas complejas.
  • Gestión bibliográfica (BibTeX).
  • Documentos largos (Tesis, Informes).
  • Consistencia tipográfica absoluta.

$$\int_{a}^{b} f(x)dx$$

Anatomía de un archivo .tex


% --- PREÁMBULO ---
\documentclass{article}
\usepackage[utf8]{inputenc}
\usepackage{amsmath}

% --- CUERPO ---
\begin{document}
    ¡Hola Mundo de Ingeniería!
\end{document}

Todo lo que va antes del "begin" define el comportamiento del documento.

Taller: Mi Primer Documento

Copia, pega y compila este código:


\documentclass[12pt]{article}
\usepackage[spanish]{babel}

\begin{document}
    Este es mi primer documento en LaTeX. 
    Usamos "article" para documentos cortos.
\end{document}

Tip: Usa report si estás escribiendo una tesis (permite capítulos).

Portada Automática

En el preámbulo:


\title{Reporte de Laboratorio 01}
\author{Juan Pérez}
\date{\today}

En el cuerpo:


\begin{document}
    \maketitle
    ...
\end{document}

Jerarquía de Contenidos

  • \section{Introducción}
  • \subsection{Marco Teórico}
  • \subsubsection{Análisis Químico}

Para generar el índice automáticamente:

\tableofcontents

Comentarios y Espacios

  • El símbolo % sirve para notas internas.
  • LaTeX ignora espacios múltiples.
  • Una línea vacía = Nuevo párrafo.

% Este texto no aparecerá en el PDF
Esto es un párrafo.

Esto es otro párrafo nuevo.
